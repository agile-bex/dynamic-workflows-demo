## What I built

- `src/todo_app/todo_store.py` — `TodoStore` class, UI-free, no PySide6/Qt.
  - CRUD: `create`, `get`, `list`, `update`, `delete`.
  - Todo is a dict with `id` (auto-incrementing int), `title`, `done` (bool).
  - Owns a single `todos.json`; `save()` writes, `reload()`/construction loads.
  - Missing file -> empty list; corrupt JSON -> empty list; non-list JSON ->
    empty list. Never crashes on load. Recovers (overwrites) on next save.
  - CRUD methods return copies so callers can't mutate internal state.
- `tests/test_todo_store.py` — 12 tests covering each CRUD path, the
  save/reload round-trip (fresh store over the same file), valid-JSON-list
  assertion, and the missing/corrupt/non-list fallbacks. All use `tmp_path`.
